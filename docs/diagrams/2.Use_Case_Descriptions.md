# Use Case Descriptions

This document provides detailed descriptions of the primary use cases identified in the KajTrack system.

### UC1: Manage Users
* **Primary Actor**: Admin
* **Description**: Allows the administrator to add, edit, view, and delete employee accounts.
* **Preconditions**: Admin must be logged into the system.
* **Main Success Scenario**:
  1. Admin navigates to "Manage Users".
  2. Admin selects "Add User" or "Edit" on an existing user.
  3. Admin fills in/updates the user's details (Full Name, Username, Password).
  4. System validates the input and updates the database.
  5. System displays a success message.

### UC2: Manage Tasks
* **Primary Actor**: Admin
* **Description**: Allows the administrator to create tasks, set deadlines, and assign them to specific employees. They can also edit or delete existing tasks.
* **Preconditions**: Admin is logged in. Employees exist in the system.
* **Main Success Scenario**:
  1. Admin navigates to "Create Task".
  2. Admin fills in task title, description, deadline, and selects an employee from the dropdown.
  3. System saves the task and triggers a notification for the assigned employee.
  4. System redirects to the task list with a success message.

### UC3: Update Task Status
* **Primary Actor**: Employee
* **Description**: Allows an employee to update the progress/status of a task assigned to them.
* **Preconditions**: Employee is logged in and has tasks assigned.
* **Main Success Scenario**:
  1. Employee navigates to "My Tasks".
  2. Employee clicks "Edit" on a specific task.
  3. Employee changes the status (e.g., Pending, In Progress, Completed).
  4. System updates the task record in the database.

### UC4: View Notifications
* **Primary Actor**: Admin, Employee
* **Description**: Users can view alerts regarding newly assigned tasks or system updates.
* **Preconditions**: User is logged in.
* **Main Success Scenario**:
  1. User clicks the Notification bell icon.
  2. System fetches unread notifications.
  3. System displays a dropdown list of recent alerts.
