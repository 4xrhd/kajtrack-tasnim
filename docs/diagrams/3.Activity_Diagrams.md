# Activity Diagrams

Activity diagrams map out the procedural flow of control from one activity to another within the KajTrack system.

## 1. Login Authentication Flow
```mermaid
flowchart TD
    Start([User navigates to Login Page]) --> EnterCreds[Enter Username and Password]
    EnterCreds --> Submit[Submit Credentials]
    Submit --> Validate{System validates credentials}
    
    Validate -- Invalid --> ShowError[Display Error Message]
    ShowError --> EnterCreds
    
    Validate -- Valid --> CheckRole{Check User Role}
    
    CheckRole -- Admin --> AdminDash[Redirect to Admin Dashboard]
    CheckRole -- Employee --> EmpDash[Redirect to Employee Dashboard]
    
    AdminDash --> End([Session Active])
    EmpDash --> End
```

## 2. Task Assignment Flow (Admin)
```mermaid
flowchart TD
    Start([Admin Dashboard]) --> ClickCreate[Click 'Create Task']
    ClickCreate --> FillForm[Fill Task Title, Description, Due Date]
    FillForm --> SelectEmp[Select Employee from Dropdown]
    SelectEmp --> SubmitTask[Submit Task]
    
    SubmitTask --> ValidateInput{Input Valid?}
    ValidateInput -- No --> ShowError[Show Error message]
    ShowError --> FillForm
    
    ValidateInput -- Yes --> SaveDB[Save Task to Database]
    SaveDB --> GenerateNotif[Generate Notification for Employee]
    GenerateNotif --> RedirectList[Redirect to 'All Tasks' List]
    RedirectList --> End([Flow Complete])
```

## 3. Update Task Status Flow (Employee)
```mermaid
flowchart TD
    Start([Employee Dashboard]) --> ViewTasks[Navigate to 'My Tasks']
    ViewTasks --> SelectTask[Click 'Edit' on specific task]
    SelectTask --> UpdateStatus[Change Status from Dropdown]
    UpdateStatus --> SubmitUpdate[Submit Update]
    
    SubmitUpdate --> SaveStatus[Save new status to Database]
    SaveStatus --> RedirectMyTasks[Redirect to 'My Tasks']
    RedirectMyTasks --> End([Flow Complete])
```
