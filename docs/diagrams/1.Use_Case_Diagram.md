# Use Case Diagram

The Use Case Diagram provides a high-level visual representation of the KajTrack system, illustrating the interactions between the system's actors (Admin and Employee) and the various use cases (features) they can execute.

```mermaid
usecaseDiagram
  actor Admin
  actor Employee

  package KajTrack System {
    usecase "Login / Logout" as UC_Login
    usecase "View Dashboard" as UC_Dashboard
    usecase "Manage Users (CRUD)" as UC_ManageUsers
    usecase "Create & Assign Tasks" as UC_CreateTask
    usecase "Manage All Tasks" as UC_ManageAllTasks
    usecase "View Assigned Tasks" as UC_ViewMyTasks
    usecase "Update Task Status" as UC_UpdateTask
    usecase "Manage Profile" as UC_ManageProfile
    usecase "View Notifications" as UC_Notifications
  }

  Admin --> UC_Login
  Admin --> UC_Dashboard
  Admin --> UC_ManageUsers
  Admin --> UC_CreateTask
  Admin --> UC_ManageAllTasks
  Admin --> UC_Notifications

  Employee --> UC_Login
  Employee --> UC_Dashboard
  Employee --> UC_ViewMyTasks
  Employee --> UC_UpdateTask
  Employee --> UC_ManageProfile
  Employee --> UC_Notifications
```
