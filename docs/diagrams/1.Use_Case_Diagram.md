# Use Case Diagram

The Use Case Diagram provides a high-level visual representation of the KajTrack system, illustrating the interactions between the system's actors (Admin and Employee) and the various use cases (features) they can execute.

```mermaid
flowchart LR
    %% Actors
    Admin([Admin])
    Employee([Employee])

    %% System Boundary
    subgraph KajTrack System
        direction TB
        UC_Login(Login / Logout)
        UC_Dashboard(View Dashboard)
        UC_ManageUsers(Manage Users)
        UC_CreateTask(Create & Assign Tasks)
        UC_ManageAllTasks(Manage All Tasks)
        UC_ViewMyTasks(View Assigned Tasks)
        UC_UpdateTask(Update Task Status)
        UC_ManageProfile(Manage Profile)
        UC_Notifications(View Notifications)
    end

    %% Admin Interactions
    Admin --> UC_Login
    Admin --> UC_Dashboard
    Admin --> UC_ManageUsers
    Admin --> UC_CreateTask
    Admin --> UC_ManageAllTasks
    Admin --> UC_Notifications

    %% Employee Interactions
    Employee --> UC_Login
    Employee --> UC_Dashboard
    Employee --> UC_ViewMyTasks
    Employee --> UC_UpdateTask
    Employee --> UC_ManageProfile
    Employee --> UC_Notifications
```
